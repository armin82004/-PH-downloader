# 🌐 Internet Downloader Suite

ابزار جامع دانلود از اینترنت، شامل دانلود ویدیو از یوتیوب و ذخیره صفحات وب به فرمت MHTML، با قابلیت اجرا از طریق GitHub Actions.

## ✨ امکانات

- 🎬 **دانلود از یوتیوب** - دانلود ویدیو، صدا و پلی‌لیست با کیفیت‌های مختلف
- 📄 **ذخیره صفحات MHTML** - ذخیره کامل صفحات وب به همراه تمام منابع (CSS, JS, Images) در یک فایل
- 🤖 **اجرا با GitHub Actions** - بدون نیاز به سرور شخصی، کاملاً خودکار
- 🔄 **پشتیبانی از اینترنت ملی ایران** - بهینه‌سازی شده برای شرایط شبکه ایران

## 📋 پیش‌نیازها

- یک حساب کاربری GitHub
- دسترسی به اینترنت (برای اجرای اکشن)
